Local Hub

Bienvenue sur l'API de localhub

URI available:
  • /login: POST json
  • /upload-avatar: POST Multipart Form containing <input type="file" name="avatar">
  • /seeder/user/<n> GET, seeds database with <n> users, returns an json array containing created emails addresses
  • /seeder/project/<user_id>/<place>/<n> GET, seeds database with <n> projects around location <place> from author <user_id>, returns an json array containing created project objects including 2 needs for each project
  • /graphql : POST graphql queries
    • Schema
    • projects queries list
    • users queries list
    • favorites queries list
    • needs queries list
